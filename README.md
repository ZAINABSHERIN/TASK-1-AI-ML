# Titanic Data Cleaning and Preprocessing

This project is part of my AI & ML Internship Task 1.

## Objective
To clean and preprocess raw data for Machine Learning.

## Dataset
Titanic Dataset

## Steps Performed
- Imported and explored the dataset
- Checked missing values and data types
- Handled missing values using median and mode
- Removed unnecessary column Cabin
- Encoded categorical columns
- Standardized numerical columns
- Detected outliers using boxplot
- Removed outliers using IQR method
- Saved the cleaned dataset

## Tools Used
Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn
